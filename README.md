# Clinic Frontend (React)

Frontend application for the Clinic Management System.

## Tech Stack
- React
- Vite
- JavaScript
- CSS

## Features
- Modern responsive UI
- API-based communication with backend
- Clean and maintainable code structure

## Backend
This frontend consumes APIs from a Django REST backend.
